CSS, ¿qué es?

El CSS(hojas de estilo en cascada) es un lenguaje que define la apariencia de un documento escrito en un lenguaje de marcado (por ejemplo, HTML ).

Así, a los elementos de la página web creados con HTML se les dará la apariencia que se desee utilizando CSS:

  • Colores
  • Espacios entre elementos
  • Tipos de letra

Esta separación entre la lec estructura y la presentación es muy importante, ya que sólo cambiando los CSS se modifique completamente al aspecto de una página web. posibilita, entre otras, cosas los usuarios pueden usar hojas de estilo perzonalizadas (como hojas de estilo de alto contraste o de accesibilidad).
